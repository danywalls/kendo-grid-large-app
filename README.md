# Super Bowl Viewers Dashboard

A demo application showcasing how to handle millions of rows efficiently using the **Kendo UI for Angular Grid**. This project implements row virtualization and server-side data fetching (endless scrolling) with Angular Signals.

**Related Article:** This code is related to articles published at [Telerik Blogs](https://www.telerik.com/blogs/author/dany-paredes) by Dany Paredes.

## Running the Project

1. Install dependencies: `npm install`
2. Activate your Kendo UI license (if you have one): `npx kendo-ui-license activate`
3. Start the development server: `ng serve`
4. Navigate to `http://localhost:4200` to see the live data feed in action.
